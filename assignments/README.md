## Adding New Assignments

When adding a new assignment:

1. Create a new folder with a clear name (e.g., `assignment_03`, `data_structures_project`)
2. Copy the template from `../templates/` if applicable
3. Add a README.md describing the assignment
4. Include all necessary files and resources
5. Update this file to list the new assignment

## Folder Naming Convention

- Use lowercase with underscores: `assignment_01`
- Or use descriptive names: `sorting_algorithms`, `web_scraper_project`
- Keep names concise but descriptive
