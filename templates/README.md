# Assignment Templates

This directory contains templates for different types of assignments to help you get started quickly.

## Available Templates

### Programming Assignment Template
Use `programming_assignment_template.md` for coding assignments.

### Report Template
Use `report_template.md` for written assignments and reports.

## Using Templates

1. Copy the appropriate template to your assignment folder
2. Rename it to `README.md`
3. Fill in the specific details for your assignment
4. Add your work files to the same folder

## Creating New Templates

Feel free to add new templates for different assignment types as needed.
